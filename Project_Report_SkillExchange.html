1. The Problem Statement

Village technicians (plumbers, electricians) often need each other's help but lack cash to pay. They can "Barter" their skills but don't know who has what skill. There is no structured platform that allows rural workers to find, negotiate, and track skill-based exchanges without monetary transactions.

2. Detailed Description (The Vision)

Skill-Exchange is a "Barter Economy" platform for rural technicians. It allows artisans to "Trade Hours" — a plumber can fix an electrician's pipes in exchange for motor repair. It builds a self-reliant community of "Skill-Swap" experts by enabling fair, transparent, and trackable skill-based exchanges using a digital platform.

The app uses Firebase for real-time data sync, a Room local database for offline support, and an AI-powered barter matching engine to suggest the best skill swap partners.

3. App Usage & User Flow

  • Skill Profile: "I am an Expert Carpenter." – Users register with their skill set.
  • Need Post: "Need help with leaking roof." – Post what skill you require.
  • Swap Offer: "I'll do your woodwork in exchange for roof repair." – Propose a barter.
  • Trust Score: Based on successful community swaps, users build a reputation score.
  • Chat: Real-time messaging to negotiate and confirm swap details.
  • Wallet: Track skill points earned and spent (1 hour = 1 point).

9. Setup & Run Instructions

  1. Clone: git clone https://github.com/Hrushikesh-hub/Skill_Exchange.git
  2. Open in Android Studio (Hedgehog 2023.1.1 or newer)
  3. Add your own google-services.json in the app/ folder
  4. Enable Firebase Auth (Email/Password), Firestore, and Realtime Database
  5. Sync Gradle and run: ./gradlew installDebug or use Run → Run 'app'
